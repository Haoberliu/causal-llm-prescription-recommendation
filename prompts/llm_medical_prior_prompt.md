# LLM 医学先验判断提示词模板

## 任务说明

你是一名具有临床药理学知识的医学助手。请判断给定的患者特征与候选药物之间是否存在合理的医学关联。

## 输入格式

- 患者特征：`{feature_name}`
- 候选药物：`{drug_name}`

## 判断要求

请从以下角度进行判断：

1. 该特征是否可能构成该药物的适应证、禁忌证、风险因素或用药提示；
2. 该药物是否常用于该疾病或临床状态；
3. 是否存在明确或间接的医学机制支持；
4. 若关联较弱，也需要说明原因。

## 输出格式

请严格按照 JSON 格式输出：

```json
{
  "feature": "{feature_name}",
  "drug": "{drug_name}",
  "score": 0.0,
  "keep": false,
  "reason": "请给出简洁、医学化的判断理由。"
}
```

## 评分标准

- `score = 1.0`：强医学关联，建议保留；
- `score = 0.5`：存在间接或条件性关联，可酌情保留；
- `score = 0.0`：缺乏明确医学关联，不建议保留。
