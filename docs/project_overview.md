# 项目说明

## 论文题目

融合因果推断与大模型的处方推荐技术实现

## 研究目标

本项目旨在构建一个面向临床场景的处方推荐系统。系统综合利用电子健康记录数据、因果结构学习方法和大语言模型医学知识，提升处方推荐结果的可解释性与临床合理性。

## 技术路线

1. 数据预处理：基于 MIMIC-IV 构建患者特征与药物标签。
2. 因果发现：使用 PC 算法挖掘特征与药物之间的候选因果关系。
3. 医学先验：使用大语言模型判断候选特征—药物边的医学合理性。
4. 掩码融合：融合 PC 结果、LLM 医学先验和人工规则，生成最终因果掩码。
5. 模型训练：基于因果掩码构建处方推荐模型。
6. 实验评估：使用多标签分类指标进行模型对比与消融分析。

## 主要实验对比

- Vanilla Linear
- Vanilla MLP
- PC-only Masked Linear
- Final Causal Masked Linear
- LLM zero-shot / one-shot / CoT prompt baseline
